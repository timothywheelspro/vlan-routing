## Questions to Ask

### 1. Filing Form
- [ ] "What form do I use for a motion to modify probation/community service?"
- [ ] "Is there a specific form number or name?"
- [ ] "Where can I get the form? (Online/at courthouse/by mail)"

### 2. Filing Deadline
- [ ] "What's the deadline to file before the January 20 court closure?"
- [ ] "How many days before the service deadline do I need to file?"
- [ ] "Is there a specific cutoff date?"

### 3. Hearing Requirements
- [ ] "Do I need a hearing for this motion?"
- [ ] "If yes, how do I schedule it?"
- [ ] "What's the timeline for hearing scheduling?"

### 4. Documentation Requirements
- [ ] "What medical documentation is required?"
- [ ] "Do I need a doctor's letter, or will VA crisis documentation work?"
- [ ] "What format should exhibits be in? (PDF, original, copies)"
- [ ] "How many copies do I need to file?"

### 5. Service Requirements
- [ ] "Do I need to serve the Commonwealth Attorney?"
- [ ] "What's the service deadline?"
- [ ] "How do I serve them? (Mail, in-person, email)"

### 6. Filing Location
- [ ] "Where do I file the motion? (Clerk's office location)"
- [ ] "Can I file by mail, or do I need to come in person?"
- [ ] "What are the office hours?"
