## üìù Filling Out the Template: Step-by-Step

### Step 1: Fixed Expenses (30 minutes)

Start with fixed expenses - they're easiest because they're the same every month:

1. **Open your bank/credit card statements**
2. **Go through each month** and identify fixed expenses
3. **Fill in the template** section by section:
   - Housing
   - Utilities (average if they vary)
   - Transportation (fixed parts)
   - Insurance
   - Debt payments
   - Business expenses

**Strategy**: List the expense, find the amount from statements, note the due date, identify which account it's paid from.

### Step 2: Variable Expenses (20 minutes)

For variable expenses, you need to calculate averages:

1. **Gather 3 months of data**
2. **Add up each category** for 3 months
3. **Divide by 3** to get monthly average
4. **Fill in the template**

**Example for Groceries:**
```
Month 1: $420
Month 2: $380
Month 3: $400
Average: ($420 + $380 + $400) √∑ 3 = $400/month
```

### Step 3: Subscriptions (30 minutes)

This requires checking multiple sources:

1. **Bank statements** - Look for recurring charges
2. **Credit card statements** - Check all cards
3. **App stores** - iOS Settings ‚Üí Subscriptions, Google Play Subscriptions
4. **Email search** - Search for "subscription", "renewal", "billing"
5. **Fill in subscription table** - Be thorough!

### Step 4: Calculate Totals (10 minutes)

1. **Add up each section** (subtotals)
2. **Calculate grand total** of all expenses
3. **Verify your math**
4. **Compare to income** (from income section)

## ‚ö†Ô∏è Common Mistakes to Avoid

### ‚ùå Mistake 1: Forgetting Annual/Quarterly Expenses
**Wrong:** Only counting monthly expenses  
**Right:** Divide annual expenses by 12, quarterly by 3

**Example:** Property tax $2,400/year = $200/month

### ‚ùå Mistake 2: Not Averaging Variable Expenses
**Wrong:** Using just one month's grocery bill  
**Right:** Average 3-6 months

**Why:** Expenses vary month to month - averaging gives accurate picture

### ‚ùå Mistake 3: Missing Hidden Subscriptions
**Wrong:** Only counting obvious subscriptions  
**Right:** Check ALL accounts, app stores, email

**Common missed:** App subscriptions, cloud storage, domain renewals, software subscriptions

### ‚ùå Mistake 4: Double-Counting Expenses
**Wrong:** Counting insurance deducted from paycheck AND as separate expense  
**Right:** Only count it once (usually as reduction in take-home pay, not as expense)

### ‚ùå Mistake 5: Not Including All Accounts
**Wrong:** Only checking primary checking account  
**Right:** Check ALL accounts (checking, savings, all credit cards, PayPal, etc.)

### ‚ùå Mistake 6: Using "Ideal" Instead of "Actual"
**Wrong:** "I should spend $300/month on groceries"  
**Right:** Use what you ACTUALLY spent (average of last 3 months)

**Why:** Budget based on reality, not goals. Goals come later in planning phase.

## ‚ùì Frequently Asked Questions

**Q: What if I can't find a bill amount?**  
A: Estimate based on similar months, or contact the company. Better to estimate than skip.

**Q: Should I include savings/investments as expenses?**  
A: No, those are transfers to yourself. Include in a separate "savings" section if tracking.

**Q: What about one-time expenses?**  
A: Only include if they're regular (like annual fees). True one-time expenses go in a separate category.

**Q: How do I handle shared expenses (with roommate/spouse)?**  
A: Only include your portion. If you pay $600 rent and split with roommate, include $300 if they reimburse you, or $600 if you pay it all.

**Q: What if expenses vary a lot month to month?**  
A: Average more months (6 months instead of 3) for more accuracy. Or use the highest month as worst-case planning.

**Q: Should I include taxes as an expense?**  
A: No, if using after-tax income (take-home pay). Taxes are already deducted. If using gross income, then include taxes.
