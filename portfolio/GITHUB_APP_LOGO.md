# GitHub App Logo Guide

**Requirements:**
- Size: 512x512 pixels (or square, at least 200x200)
- Format: PNG, JPG, or SVG
- File size: Under 1MB

## How to Use

### Option A: Use the SVG Files Directly

1. **Convert SVG to PNG** (GitHub may prefer PNG):
   ```bash
   # Using ImageMagick (if installed)
   convert github-app-logo.svg -resize 512x512 github-app-logo.png
   
   # Or use an online converter:
   # https://cloudconvert.com/svg-to-png
   ```

2. **Upload to GitHub:**
   - Go to your GitHub App settings
   - Find "Display information" section
   - Click "Upload" next to "Logo"
   - Select your logo file

### Option B: Use Online Tools

**Quick logo generators:**
- https://www.canva.com/ (free, easy)
- https://www.logomaker.com/ (free)
- https://hatchful.shopify.com/ (free)

**Create a simple logo:**
- Text: "CYW" or "Contruil"
- Background: Dark (#0f1620 or #0b0f14)
- Text color: Cyan (#3de0ff)
- Size: 512x512px
- Export as PNG

### Option C: Use Your Existing Branding

If you have a logo already:
- Resize to 512x512px
- Ensure it's square
- Save as PNG or JPG
- Upload to GitHub

## Logo Specifications

**GitHub Requirements:**
- ✅ Square (1:1 aspect ratio)
- ✅ Minimum 200x200px (512x512px recommended)
- ✅ PNG, JPG, or SVG format
- ✅ Under 1MB
- ✅ Clear at small sizes (will be shown at 64x64px in some places)

**Your Brand Colors:**
- Background: `#0b0f14` or `#0f1620` (dark)
- Accent: `#3de0ff` (cyan)
- Text: `#eaf2ff` (light)
